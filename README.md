# üìú How to use
How to use PKitty:

1) - Extract the ZIP File.

2) - Open the file "mainPK.bat"

3) - Enter the IPv4 Address you want to scan for ports (E.G,1.1.1.1)
   - (If the IP is offline, Scan a different IP)
